HTML

Okay, now the good stuff. This a reference for HTML5. Here's a template for a basic html page.

Main Layout

Some tags you'll need for the main layout of the web page.

Tag Usage
<-- !--> Comments in HTML
<!DOCTYPE> Defines the document type.
<html> Defines the root of the page
<head> Contains metadata of the page
<title> The title is displayed in the web browser’s tab. Place it in the head tag
<main> The main area of the page
<body> Body of the page – usually in the main tags
<header> This will be a header on the page
<nav> The navigation bar of a page
<section> Defines a section – an area of the main page
<figure> The figure tag is usually only one element i.e a picture or diagram. Figcaption can be used to add text to the figure
<div> Style tags can be used to apply CSS styling straight into the HTML file. It’s not recommended for complex styling.
<script> This tag can be used to add Javascript to an HTML file.
<footer> The footer section of the page

Content Tags

These tags will allow you to add content to your webpage

Tag Usage
<a href=” ”>; A link tag. The link/path should go inside the quotations. You can wrap this around images or text to create clickable text/pictures
<img src=””> The tag to insert images. The link/path to the image will go inside the quotations. You can also add width and height properties.

Tables

Here is a reference for table tags.

Tag Usage
<table> This creates a table
<th> This creates a heading cell
<tr> This creates a normal row
<td> This creates a table cell.

Forms

Tags to create forms

Tag Usage
<form> The form tag is used to creat a form for input. The input and labels should go inside this tag. It usually has action and method attributes.
<input> This creates the input box that the user will enter. The input has three attributes: type, name, and id. There are several different type attributes: text, radio, checkbox, submit, button, etc.
<label> The label tag creates a label for the input. The attribute "for" should be the same as the input's id to join them.

CSS

CSS can style web pages, and can even do simple animation. Here are some basic tips.

Syntax

There several parts to the CSS syntax: selectors, properties, and values.

Selectors

HTML tags in CSS become the selectors - they become the areas styles are applied to. For example, if you wanted to make the font colour of the nav red, it would look like this:

HTML Tag CSS Selector Notes
<tagName> tagName{ } To select areas that are regular HTML tags e.g. nav, main, etc.
<tagName class="className"> .className { } Classes are an attribute used to point to certain sections in the HTML. It can be used by both CSS and JavaScript
<tagName id="tagName"> #idName {} Ids are also an attribute used to point to a section in the HTML. Unlike classes they are usually used only once.

Properties and Values

Properties are the thing that you want to style. Values are what the property is set as. In the example above, the property is the color (the font colour) and it has been set to red.
The list of properties and their values is pretty exhaustive, but this table goes through some common ones. I recommend W3C Schools or Mozilla Development Notes if you are looking for something really specific.

Property Description Possible Values
margin Sets a margin around the outside of an element Can be set using px or percentage. Different values can be specified: top, right, bottom, left. Auto allows the browser to set a suitable margin.
padding Padding is the space within an element. Can be set using px or percentage. Different values can be specified: top, right, bottom, left.
colour The colour of the text Values can be entered using words (e.g. black), rgb, or hexadecimal codes.
font-size Changes the font size Can be entered as a percentage, em, or in pixels
font-family Changes the font Several fonts can be used. If one is not available, then the next on the list will be used. Fonts with more than one word go in single quotes.
text-align Changes the alignment of text Left, right, center, justify are the main values
line-height Changes the heigh of line boxes within an element Can be specified as em, percentage, or pixels.

The Box Model

The CSS box wraps around HTML elements. There are several layers to the elements:

The Flexbox

In the box model, there are only really 4 types of positioning - block, inline, table, absolute. As you can imagine, that's not very flexible. That's where flexbox comes in.
The flexbox model allows different positioning of items within a flex container.
There are many ways to position items, which can be read about on W3Schools.

Here are some necessary flexbox properties and values:

Property Description Possible Values
display Sets the display. For flexbox, this should be used on the container of the items. The value should be flex or inline-flex.
flex-direction Specifies the order of the flex items row, column, and row/column-reverse
flex-wrap Items will not wrap in the container unless this value is uesd nowrap (default), wrap, wrap-reverse
justify-content Aligns items along the main axis flex-start, flex-end, center, space-between, space-around, space-evenly
